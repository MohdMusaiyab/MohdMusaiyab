## hi, I'm Musaiyab

Full stack developer — I build web apps end to end, from database to UI. My go-to stack is **MERN + Next.js**, and I've been picking up **Go** for backend work that needs to be fast and solid.

Lately I've been spending a lot of time on two things — figuring out how to integrate AI into full stack products in ways that actually make sense, and going deeper into **system design**. How things scale, how services talk to each other, where things break. That stuff genuinely interests me.

I also like going back to CS fundamentals. The more I build, the more I realise how much the basics matter.
